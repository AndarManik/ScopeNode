# Scope Node

**Scope Node** is a multiplayer top-down shooter which expores the essence of positioning in raycast combat. Players control turret drones which both automatically aims and instantly kills enemies.

Besides killing all opponents, a round win can be obtained by being the first team to touch the orb in the center of the map. 

The arena is initially populated with random obstacles and after every round, the loseing team gets to place an obstacle at a choosen position.

Matches are decided once a team wins N rounds.
